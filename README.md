just for practice, if you want to use real ups app, go to [UPS ios](https://apps.apple.com/tw/app/ups-mobile/id336377331) :)

## Quick Start

install package and dependencies

```bash
yarn install
```

start tailwind converter

```bash
yarn dev:tailwind
```

start app

```bash
yarn start
```
